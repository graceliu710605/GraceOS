## 模块职责

### M1 首页仪表盘
- 数字健康评分（0-100）
- 关键指标卡片（文件数/C盘/软件数/重复率）
- Top 问题列表
- 快速入口

### M2 数字健康中心
- 健康评分明细（各维度得分与扣分原因）
- 评分历史趋势
- 问题详情入口

### M3 软件健康中心
- 多版本软件检测
- 重复软件检测
- 长期未更新软件
- 建议保留/删除列表

### M4 存储空间分析
- 按目录统计空间占用
- Downloads/Desktop/Documents/WeChat/Projects/Obsidian
- 空间占用排行榜

### M5 自动备份中心
- Git 仓库自动 commit + push
- SQLite 数据库定时备份
- 配置文件备份（VSCode/Codex/Obsidian）
- 备份历史日志

### M6 项目管理中心
- 项目目录扫描与发现
- Git 状态（分支/未提交/未推送）
- 项目最后修改时间
- 项目大小统计

### M7 知识库中心
- Obsidian vault 统计
- Markdown 数量/附件数量
- 目录结构浏览

### M8 Prompt 中心
- Prompt 分类管理
- 搜索/标签/收藏
- 版本记录

### M9 数字资产周报
- Markdown 格式自动报告
- 软件/文件/空间/健康分变化
- 每周自动生成

## 数据流

```
Scanners → JSON files → SQLite import → Service Layer → Dashboard
                                              ↓
                                        Scoring Engine
                                              ↓
                                    AI Suggestions (future)
```

## 模块耦合

| 上游模块 | 下游模块 | 数据依赖 |
|---------|---------|---------|
| 文件扫描器 | 健康评分 | 重复文件数、未使用文件数 |
| 磁盘扫描器 | 健康评分 | C盘使用率 |
| 软件扫描器 | 软件健康 | 版本信息、安装日期 |
| 所有模块 | 周报 | 汇总统计 |
