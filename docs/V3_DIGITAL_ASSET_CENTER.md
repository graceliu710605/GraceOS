# V3 数字资产中心 — 需求与 UI 设计

**版本**: V3 Alpha
**日期**: 2026-06-13
**状态**: 仅需求文档，UI 待确认，禁止开发

---

## 功能定位

管理个人互联网资产。V3 第一阶段：纯 URL 收藏+管理，不接 API，不抓数据，不做同步。

## 资产分类

| 分类 | 示例 |
|------|------|
| 内容平台 | 小红书/B站/YouTube/抖音/视频号 |
| AI 平台 | ChatGPT/Claude/DeepSeek |
| 开发平台 | GitHub |
| 云服务 | 腾讯云/阿里云/七牛云/Cloudflare |
| 域名 | 个人域名 |
| 其它 | 自定义 URL |

## 第一阶段功能

- 打开网址（一键跳转浏览器）
- 新增资产（名称/URL/分类）
- 编辑资产
- 删除资产
- 按分类筛选
