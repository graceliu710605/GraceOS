## 技术风险

- **删除操作不可逆**: 文件删除移至 `E:\回收站`（非系统回收站），用户需手动管理
- **软件卸载依赖 winget**: Windows 10/11 自带，但部分系统可能未安装
- **软件启动**: 仅能在 install_path 中找到 .exe 时启动，路径为空时无法启动
- **Streamlit 原生限制**: 无法实现真正的 Tab URL 路由，使用 session_state 实现导航
