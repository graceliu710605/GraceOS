# GraceOS V2 Beta Release Notes

> 发布日期: 2026-06-12
> Commit: 57cec66

## 版本定位

GraceOS V2 Beta — **个人数字资产管理系统**。从资产分析工具升级为资产管理平台。

## 核心功能

### 🏠 首页仪表盘
- 数字健康评分 0-100
- AI 建议中心（4维自动分析）
- 评分历史趋势

### 📂 文件资产中心
- 全盘文件索引 (115万+)
- 文件搜索
- **重复文件处理中心**: 保留/删除/归档 + 空间预估
- **未使用文件处理中心**: 删除/归档/打开 + 系统保护
- Top 100 大文件

### 📋 软件资产中心
- 已安装软件清单 (142个)
- **点击名称打开安装目录** (36个有路径)
- **多版本软件处理**: 检测 + 建议保留最新
- 搜索 + 排序

### 📑 磁盘资产中心
- C/D/E 盘容量 + 使用率颜色 (绿/黄/红)
- C 盘空间预警 (>80%)
- 点击盘符打开目录

## 产品化

- ✅ Start_GraceOS.bat 一键启动
- ✅ 首次运行自动初始化
- ✅ 浏览器自动打开
- ✅ 安装指南 + 文档齐全
