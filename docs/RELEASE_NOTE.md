# GraceOS V2 Beta Release Notes

> 发布日期: 2026-06-12 | 版本: V2 Beta | Commit: `(pending)`

## 版本定位

GraceOS 从**资产分析工具**升级为**资产管理平台**。

核心闭环：**发现问题 → AI建议 → 一键执行**

## P0 功能

### ✅ 1. 修复软件安装路径获取
- 修复 PowerShell 注册表查询的 JSON 编码问题
- 添加 InstallLocation 字段到 registry 扫描
- 点击软件名称 → 直接打开安装目录

### ✅ 2. 重复文件处理中心
- **推荐保留**: 自动识别原始文件（最早路径）
- **推荐删除**: 识别副本文件（重复路径）
- **预计释放空间**: 实时计算可释放空间
- **回收站删除**: 移至专用回收站目录（安全删除）
- **归档**: 移至归档目录保留结构

### ✅ 3. 多版本软件处理中心
- 检测多版本软件（GROUP BY name HAVING COUNT > 1）
- **推荐保留版本**: 最新版本
- **推荐删除版本**: 最旧版本
- 一键在 winget 中搜索（辅助手动卸载）

### ✅ 4. 长期未使用文件处理中心
- 显示前100个最久未修改文件
- 自动排除系统文件（Windows/System32/pagefile/hiberfil）
- **删除**: 移至回收站
- **归档**: 移至归档目录
- **打开目录**: 快速定位文件

## P1 功能

### ✅ 5. 系统保护白名单
- 自动排除: C:/Windows, System32, pagefile.sys, hiberfil.sys
- 未使用文件扫描自动跳过系统路径

### ✅ 6. 删除预演
- 重复文件：显示"保留一份可释放约 X MB"
- 未使用文件：显示"共 X MB"
- 操作前确认：选择文件后可预览详细信息

### ✅ 7. AI 建议中心
- 首页自动分析四大维度问题
- **发现问题** → **推荐处理** → **一键执行**
- 每个建议含：图标、标题、说明、预计收益、快速跳转
