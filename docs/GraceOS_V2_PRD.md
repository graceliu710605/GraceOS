# GraceOS V2 产品需求文档

> GraceOS = 个人数字操作系统（Personal Digital Operating System）
> 不是文件管理器。不是清理工具。而是统一数字资产管理平台。

## 产品定位

**GraceOS** 帮助 AI 重度用户、产品经理、程序员、创业者统一管理 7 类数字资产：

| 资产类型 | 说明 |
|---------|------|
| 文件资产 | 全盘文件索引、搜索、重复检测 |
| 软件资产 | 已安装软件、版本管理、健康分析 |
| 磁盘资产 | 磁盘容量、使用率监控 |
| 知识资产 | Obsidian 知识库、文档搜索 |
| Prompt 资产 | Prompt 分类、版本、复用 |
| 项目资产 | Git 项目、代码仓库管理 |
| 自动化资产 | n8n 工作流、定时任务 |

## 核心价值

**V1 → V2 转变**：从"看清问题"到"评估问题"到"解决问题"

1. **健康评分**：0-100 分量化数字资产健康度
2. **AI 建议**：自动识别问题并给出可执行建议
3. **一键执行**：从建议到执行形成闭环
4. **持续监控**：周报跟踪变化趋势

## 目标用户

- AI 重度用户（Cursor/Codex/ChatGPT 日常使用）
- 产品经理（多项目并行，文档分散）
- 程序员（多 Git 仓库，磁盘管理痛点）
- 自由职业者/创业者（个人资产管理需求）

## V2 模块总览

| 编号 | 模块 | 优先级 | 说明 |
|------|------|--------|------|
| M1 | 首页仪表盘 | P0 | 健康评分 + 关键指标 |
| M2 | 数字健康中心 | P0 | 评分明细 + 问题列表 |
| M3 | 软件健康中心 | P1 | 多版本/重复/过期软件 |
| M4 | 存储空间分析 | P1 | 目录级别空间统计 |
| M5 | 自动备份中心 | P1 | Git/SQLite/Config 备份 |
| M6 | 项目管理中心 | P2 | Git 项目列表与状态 |
| M7 | 知识库中心 | P2 | Obsidian 统计与搜索 |
| M8 | Prompt 中心 | P2 | Prompt 资产管理 |
| M9 | 数字资产周报 | P2 | Markdown 自动报告 |
| M10 | 设置中心 | P2 | 扫描/备份/通知配置 |

## 开发策略

**向前兼容**：V1 功能保持不变
**先做 P0**：健康评分 + 软件健康 → 立即产生用户价值
