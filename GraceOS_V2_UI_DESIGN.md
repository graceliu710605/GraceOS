# GraceOS V2 页面设计

## 设计原则

1. **表格为主**：继承 V1 表格模式，所有列表使用 st.dataframe
2. **单行显示**：每条记录一行，支持横向滚动
3. **渐进展示**：先看概览，再看明细，最后执行
4. **无大按钮**：交互统一使用行选择 + 列操作

## 整体布局

侧边栏导航 + 主内容区。V2 从 3 Tab 扩展为侧边栏多页面模式。

### 2. 资产中心（V1 升级）

**文件资产中心**：
- 继承 V1 表格模式
- 新增：按文件类型筛选（文档/图片/视频/代码等）
- 新增：按时间范围筛选
- 新增：批量选择 + 导出列表

**软件资产中心**：
- 继承 V1 表格模式
- 新增：多版本标记（同一软件多版本标黄）
- 新增：长期未使用标记（>90天无扫描记录标灰）
- 新增：可升级标记（winget 有新版本标蓝）

**磁盘资产中心**：
- 继承 V1 设计
- 新增：历史使用率趋势（最近7天/30天）
