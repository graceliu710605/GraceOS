## GraceOS 特有规则

### 测试执行

开发完成后必须执行 L1（开发测试）+ L2（Sanity Test），并提交测试证据。

**L2 Sanity Test** 覆盖：首页 / 文件搜索 / 重复文件 / Top100 / 未使用文件 / 软件中心 / 磁盘分析。

### 完整回归测试

默认不执行。由 Grace 在版本发布前人工验收。

### 发布标准
非开发人员电脑可直接安装运行。

### 产品阶段冻结机制
进入发布阶段后禁止新增功能。
