# 个人数字资产管家 开发规范

> 所有开发任务默认按本规范执行。
> 自动更新 PROJECT_STATUS.md、生成报告、提交 Git。
> 无需再次特别说明。

---

## 产品定位

个人数字资产管家是数字资产管理系统（项目代号 GraceOS）。

目标闭环：

**发现问题 → 分析问题 → 给出建议 → 执行处理**

### 优先开发

- 文件资产管理
- 软件资产管理
- 磁盘资产管理
- AI 分析建议
- 自动整理
- 资产处理能力

---

## 产品原则

❌ 避免：只统计，不处理

✅ 所有分析功能必须形成完整闭环：

**查看 → 建议 → 执行 → 处理**

---

## 冻结模块

以下模块保留菜单和页面，显示 **Future Module（待后续版本开发）**：

- 项目中心
- 知识库中心
- Prompt 中心
- 工作流中心
- 备份中心
- 设置中心

暂不继续开发，除非产品规划调整。

## AI 建议规范

对于以下资产类型，必须输出完整建议：

| 资产类型 | 问题 | 原因 | 建议 | 预计释放空间 | 推荐处理方式 |
|---------|------|------|------|-------------|-------------|
| 重复文件 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 长期未使用文件 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 多版本软件 | ✅ | ✅ | ✅ | - | ✅ |
| 大文件 | ✅ | ✅ | ✅ | ✅ | ✅ |

---

## 删除安全规范

支持三种处理模式：

| 模式 | 路径 | 说明 |
|------|------|------|
| 回收站 | `E:\回收站` | 可恢复 |
| 直接删除 | 永久 | 谨慎使用 |
| 归档 | `E:\归档` | 保留目录结构 |

执行前必须显示：

1. 预计删除文件数
2. 预计释放空间
3. 危险操作确认

---

## 白名单

支持设置**永不清理目录**：

- `E:\知识库obsidian`
- `E:\创业项目`
- 照片目录
- 家庭资料目录
- 用户自定义目录

白名单目录：不参与 AI 建议，不参与自动处理。

---

## 系统保护

禁止推荐处理以下系统路径：

`C:\Windows`, `C:\Windows\System32`, `C:\Program Files`, `C:\Program Files (x86)`, `pagefile.sys`, `hiberfil.sys`, `EFI`, `Recovery`, `Boot`

以及其它系统关键目录。

## Git 规范

每完成一个模块：

```bash
git add .
git commit -m "模块名: 变更说明"
```

如已配置远程仓库：`git push`
