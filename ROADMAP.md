## V2.0 — 智能资产管理（核心升级）
**优先级**: P0-P2 | **预计工作量**: 3-6个月

### P0：数字健康度评分系统
| 功能 | 说明 | 用户价值 |
|------|------|---------|
| 综合健康分 | 0-100 分，基于重复率/空间/未使用等多维指标 | 一眼看出电脑健康状态 |
| 重复文件评分 | 重复文件数/占用空间 | 量化空间浪费 |
| 磁盘空间评分 | C盘使用率/总容量 | 预警磁盘危机 |
| 长期未使用评分 | N天未修改文件占比 | 识别可清理文件 |
| 软件健康评分 | 多版本/过期/未知发布商 | 软件安全评估 |
| 趋势图 | 每日/每周评分变化 | 跟踪改善效果 |
| 评分明细 | 各项得分及扣分原因 | 知道如何改善 |

### P1：AI 整理建议
| 功能 | 说明 | 用户价值 |
|------|------|---------|
| 空间释放建议 | Top 可删除/可移动文件 | 一键释放N GB |
| 重复文件建议 | 保留哪个、删除哪个 | 避免误删 |
| 归档建议 | 超过N天未修改 → 建议归档 | 保持工作目录整洁 |
| 软件清理建议 | 长期未使用软件 → 建议卸载 | 减少软件膨胀 |
| 自然语言交互 | "帮我清理C盘" → AI分析+建议 | 降低使用门槛 |
| 一键执行 | 勾选建议 → 批量执行 | 从看问题到解决问题 |

### P1：自动整理中心
| 功能 | 说明 | 用户价值 |
|------|------|---------|
| 下载目录自动归档 | 规则：超过N天 → 移动到归档 | 下载目录不再堆积 |
| 微信文件归档 | 微信接收文件 → 自动分类 | 微信文件不再丢失 |
| 桌面自动整理 | 按类型归类（文档/图片/安装包） | 桌面保持整洁 |
| 自定义规则 | 用户自定义归档规则 | 满足个性化需求 |
| 预览模式 | 先预览后执行 | 安全可控 |

### P1：自动备份中心
| 功能 | 说明 | 用户价值 |
|------|------|---------|
| Git 仓库自动备份 | 检测本地 Git 仓库 → 自动 push | 代码零丢失 |
| SQLite 自动备份 | 定时备份数据库文件 | 数据安全 |
| 配置文件备份 | VSCode/Codex/Obsidian 配置 | 开发环境可恢复 |
| 备份仪表盘 | 显示所有备份状态 | 一目了然 |
| 异地备份 | 支持 GitHub/GitLab/Gitee | 多地容灾 |

### P2：项目管理中心
| 功能 | 说明 | 用户价值 |
|------|------|---------|
| 项目列表 | 自动发现本地项目目录 | 不用手动管理 |
| Git 仓库状态 | 未提交/未推送/分支 | 代码状态监控 |
| 项目大小统计 | node_modules/venv 等 | 识别膨胀项目 |
| 项目备份 | 一键打包备份 | 项目归档 |
| 最后修改时间 | 活跃度跟踪 | 判断是否可归档 |

### P2：Obsidian 知识库中心
| 功能 | 说明 | 用户价值 |
|------|------|---------|
| 知识库统计 | 笔记数/标签/链接 | 知识库概况 |
| 文档搜索 | 全文搜索 Obsidian vault | 快速查找知识 |
| 知识图谱预留 | 双向链接可视化（预留接口） | 为 V3 做准备 |
| 备份同步 | Git 自动同步 vault | 知识安全 |

### P2：Prompt 管理中心
| 功能 | 说明 | 用户价值 |
|------|------|---------|
| Prompt 资产库 | 存储/管理所有 Prompt | Prompt 可复用 |
| 分类标签 | 按用途分类 | 快速检索 |
| 版本管理 | Prompt 版本历史 | 迭代优化 |
| 使用统计 | 使用次数/效果评分 | 优化依据 |
| 导入导出 | JSON/Markdown 格式 | 跨平台使用 |

### P2：工作流中心
| 功能 | 说明 | 用户价值 |
|------|------|---------|
| n8n 工作流集成 | 查看/管理 n8n 工作流 | 自动化可视化管理 |
| 定时扫描 | 定时执行资产扫描 | 数据保持新鲜 |
| 自动报告 | 每周资产报告生成 | 定期掌握状态 |
| Webhook 触发 | 外部事件触发工作流 | 扩展自动化能力 |
