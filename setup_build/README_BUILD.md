## Quick Build (with NSIS installed)

### Prerequisites

- NSIS 3.x: winget install NSIS.NSIS
- Internet (to download Python embed ~9MB)

### Build Steps

`powershell
cd setup_build
.\build.ps1          # Download Python + install deps + copy app
makensis setup.nsi   # Produce ../个人数字资产管家_Setup.exe
`

### Verify

Follow RELEASE_CHECKLIST.md gate rules before release.
